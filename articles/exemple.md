# Mon premier article

Ceci est un exemple d'article écrit en **Markdown** pour tester le nouveau système de gestion d'articles via GitHub.

## Pourquoi ce changement ?
- Plus besoin d'éditeur intégré : tout se gère directement dans GitHub.
- Les articles sont versionnés et collaboratifs.
- Utilisation de Mistral + l'extension GitHub pour créer/modifier les articles.

## Fonctionnalités
- **Markdown** pour le contenu (titres, listes, liens, etc.).
- **Métadonnées** (titre, date, catégorie, etc.) dans le frontmatter.
- **Design** préservé pour une expérience utilisateur cohérente.

## Exemple de code
```javascript
// Exemple de code dans un article
function hello() {
  console.log("Bonjour depuis GitHub !");
}
```

> ⚠️ **Note** : Les articles sont maintenant gérés via GitHub. Utilisez Mistral + l'extension GitHub pour les modifier.
